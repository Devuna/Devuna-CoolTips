UpdateTipText (update Tip Text for the control)

     UpdateTipText(HWND hwndControl, STRING sTip, BOOL bMultiline)

 
UpdateTipText Updates the Tip Text for the control.
hwndControl This is the {prop:handle} for the window control that is associated with the tip.
sTip Specifies the tip text.
bMultiline If true, crlf is used to delimit multiple lines of text. If false, all text is displayed in one line.
Return Value Returns true if successful, or false otherwise.
