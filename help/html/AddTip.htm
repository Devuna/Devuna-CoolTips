AddTip (create a cooltips control)

     AddTip(HWND hwndControl, STRING sTip, BOOL bMultiline=0, STRING sTitle, LONG nIcon, STRING sFont, BYTE nFontSize)

 
AddTip Creates a CoolTips control.
hwndControl This is the {prop:handle} for the window control that is to be associated with the tip.
sTip Specifies the tip text.
bMultiline If true, crlf is used to delimit multiple lines of text. If false, all text is displayed in one line.
sTitle Specifies the Tip Title. The Tip Title is shown in bold at the top of the tip.
nIcon Icon Number:
  0 = None
  1 = Info
  2 = Warning
  3 = Error
sFont Specifies the name of the font to be used for the tip
nFontSize Specifies the point size for the Tip font. Must be in the rang 6 - 24.
Returns true if successful,or false otherwise.
