EnumTools (retrieve information about the ToolTip control tools)

     EnumTools(UINT iTool, *tagTOOLINFO ti)

 
EnumTools Retrieves the information that a ToolTip control maintains about the current tool.
iTool Zero-based index of the tool for which to retrieve information.
ti Address of a toolinfo structure that receives information about the tool.
Return Value Returns true if any tools are enumerated, or false otherwise.
