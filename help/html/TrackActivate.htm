TrackActivate (activates or deactivates a tracking ToolTip)

     TrackActivate(HWND hwndControl, BOOL bActivate)

 
TrackActivate Activates or deactivates a tracking ToolTip.  
hwndControl This is the {prop:handle} for the window control that is associated with the tip.  
bActivate Value specifying whether tracking is being activated or deactivated. This value can be one of the following:  
  true activate tracking.
  false deactivate tracking.
Return Value Returns true if successful, or false otherwise.
Requires version 4.70 and later of Comctl32.dll
