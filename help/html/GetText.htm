GetText (Retrieve the current ToolTip text for a control)

     GetText(HWND hwndControl, *CSTRING szText)

 
GetText Retrieves the current ToolTip text for a control.
hwndControl This is the {prop:handle} for the window control that is associated with the tip.
szText Address of a buffer to receive the text.
Return Value Returns true if successful, or false otherwise.
See Also: UpdateTipText
