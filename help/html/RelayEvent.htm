RelayEvent (pass a mouse message to a ToolTip control for processing)

     RelayEvent( *tagMSG msg)

 
RelayEvent Passes a mouse message to a ToolTip control for processing.
msg Address of an msg structure that contains the message to relay.
Return Value Returns true if successful, or false otherwise.
 
A ToolTip control processes only the following messages passed to it by the RelayEvent method:
  WM_LBUTTONDOWN
  WM_LBUTTONUP
  WM_MBUTTONDOWN
  WM_MBUTTONUP
  WM_MOUSEMOVE
  WM_RBUTTONDOWN
  WM_RBUTTONUP
  all other messages are ignored.
