AdjustRect (calculate ToolTip display rectangle)

     AdjustRect(BOOL fLarger,*tagRECT rc )

 
AdjustRect Calculates a ToolTip control's text display rectangle from its window rectangle, or the ToolTip window rectangle needed to display a specified text display rectangle.
fLarger Value that specifies which operation to perform. If true, rc is used to specify a text-display rectangle and it receives the corresponding window rectangle. If false, rc is used to specify a window rectangle and it receives the corresponding text display rectangle.
rc rect structure to hold either a ToolTip window rectangle or a text display rectangle.
Return Value Returns a non-zero value if the rectangle is successfully adjusted, and returns zero if an error occurs.
Requires version 5.80 or later of Comctl32.dll
