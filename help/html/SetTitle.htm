SetTitle (add icon and title to ToolTip)

     SetTitle(*CSTRING szTitle, WORD wIcon)

 
SetTitle Adds a standard icon and title string to a ToolTip.  
szTitle A pointer to the title string. You must assign a value to szTitle.  
wIcon Set wIcon to one of the following values to specify the icon to be displayed:  
  0 No Icon
  1 Info Icon
  2 Warning Icon
  3 Error Icon
Return Value Returns true if successful, or false otherwise.
Requires version 5.80 and later of Comctl32.dll
