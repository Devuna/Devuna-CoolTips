HitTest (test point to see if in bounding rectangle of tool)

     HitTest(HWND hwndControl, *tagPOINT pt, *tagTOOLINFO ti)

 
HitTest Tests a point to determine whether it is within the bounding rectangle of the specified tool and, if it is, retrieves information about the tool.
hwndControl This is the {prop:handle} for the window control that is associated with the tip.
pt Address of a point structure. This parameter needs to be initialized to the point in question.
ti Address of a toolinfo structure that receives information about the tool.
Return Value Returns true if the tool occupies the specified point, or false otherwise.
