Init (initialize the CoolTips class)

     Init(HWND hwndParent, BOOL bBalloonTips)

 
Init Initializes the CoolTips Class. This method must be called before making any other calls to the class.

This method registers specific common controls classes from the common control dynamic-link library (comctl32.dll) and initializes properties private to the class.
hwndParent This is the procedure window{prop:hwnd}
bBalloonTips If true, balloontips will be used if supported by the version of comctl32.dll on the users machine. If false, normal rectangular tooltips will be used.
Return Value Returns the window handle for the ToolTip control.
See Also: KCR_CoolTips
