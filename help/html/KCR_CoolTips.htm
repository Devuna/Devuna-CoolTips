KCR_CoolTip extension template

The KCR_CoolTip template adds a CoolTips control to the window.

 

 

Don't apply at all Disables code generation when checked.
Use Ballooon Tips Check this option to display balloon style tooltips if the users version of comctl32.dll supports them.
Replace regular tip Check this option to automatically replace all regular tips on controls with Cooltips.
Selected Controls This button takes you to the following screen where you can specify the details for the controls you want tool tips for.

 

Pressing the Insert or Properties buttons, displays the tooltip detail for the control: 


 

Multi-Line Tip Check this option for multi-line tool tips. Separate individual lines with crlf '<13,10>'
ToolTip Text Choose one of the options for the source of the ToolTip text:  
  Use Tip Text This option will use the tip text you specified in the IDE
  Use Local Data Variable This option will allow you to select a local data variable to be use as the source for the tip text.
  Use Global Data Variable This option will allow you to select a global data variable to be use as the source for the tip text.
  Specify Text This option will allow you to specify the tip text.
Title Options
  Title Text Enter the text to use for the tip title. This will be displayed in bold font at the top of the balloon tooltip.
  Icon Select the icon to display to the left of the title.
Font Options
  Select Font Displays the windows standard Font dialog. Only the FontName and FontSize are used.
