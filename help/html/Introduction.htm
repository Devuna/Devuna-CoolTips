CoolTips Class API and Template Specification

The following help file contains descriptions of the methods defined in the CoolTips class and descriptions of the templates provided to simplify the use of the class.

CoolTips provides a Clarion wrapper (CoolTips.clw) for the ToolTip Common Control located in the Windows Common Control Library (comctl32.dll).

To use the CoolTips class in your application follow this simple step:
     Add the KCR_CoolTips extension template to the procedure in which you wish to use the ToolTip common control.

A sample program CoolDemo.app is provided that demonstrates use of the templates supplied with the CoolTips Class and Templates.
