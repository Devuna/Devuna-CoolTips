SetTipBkColor (set the background color)

     SetTipBkColor(COLORREF clr)

 
SetTipBkColor Sets the background color in a ToolTip window.
clr New background color.
Return Value Returns true if successful, or false otherwise.
Requires version 4.70 and later of Comctl32.dll
When visual styles are enabled, this method has no effect.
See Also: GetTipBkColor
