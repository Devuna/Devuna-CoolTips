GetBubbleSize (returns the width and height of a ToolTip control.)

     GetBubbleSize(HWND hwndControl, *WORD wWidth, *WORD wHeight)

 
GetBubbleSize Returns the widht and height of a ToolTip control.
hwndControl This is the {prop:handle} for the window control that is associated with the tip.
wwidht Address of widht variable.
wHeight Address of height variable.
Return Value Returns true if successful, or false otherwise.
Requires version 5.80 and later of Comctl32.dll
