SetMargin (sets the margins for a ToolTip window)

     SetMargin(*tagRECT rc)

 
SetMargin Sets the top, left, bottom, and right margins for a ToolTip window. A margin is the distance, in pixels, between the ToolTip window border and the text contained within the ToolTip window.
rc Address of a rect structure that contains the margin information.
Return Value Returns true if successful, or false otherwise.
Requires version 4.70 and later of Comctl32.dll
See Also: GetMargin
