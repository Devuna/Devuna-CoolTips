GetToolInfo (retrieve information that ToolTip control maintains about tool)

     GetToolInfo(HWND hwndControl, *tagTOOLINFO ti)

 
GetToolInfo Retrieves the information that a ToolTip control maintains about a tool.
hwndControl This is the {prop:handle} for the window control that is associated with the tip.
ti Address of a toolinfo structure that receives information about the tool.
Return Value Returns true if successful, or false otherwise.
See Also: SetToolInfo
