TrackPosition (set the position of a tracking ToolTip)

     TrackPosition(SIGNED xPos, SIGNED yPos)

 
TrackPosition Sets the position of a tracking ToolTip.
xPos The x-coordinate of the point at which the tracking ToolTip will be displayed, in screen coordinates.
yPos The y-coordinate of the point at which the tracking ToolTip will be displayed, in screen coordinates.
Return Value Returns true if successful, or false otherwise.
Requires version 4.70 and later of Comctl32.dll
See Also: SetMaxTipwidht
