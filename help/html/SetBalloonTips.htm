SetBalloonTips (switch between Normal and Balloon style tips)

     SetBalloonTips(BOOL fBalloonTips)

 
SetBalloonTips Sets the style of tips displayed by the ToolTip control.
fBalloonTips Style flag. If this parameter is true, the ToolTip control will display Ballon Style tips. If it is false, the ToolTip control will display Normal Style tips.
Return Value Returns true.
